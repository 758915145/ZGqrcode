##二维码程程软件（震革专用）##
智能充电站插座二维码生成软件。

##使用方法##
1、安装[Node.js](http://nodejs.cn/);

2、安装[GraphicsMagick-1.3.25-Q16-win32-dll](http://www.graphicsmagick.org/);

3、下载这个仓库;

4、在cmd命令行输入：npm install cnpm -g

5、在cmd命令行输入：cnpm install

6、在cmd命令行输入：npm start
